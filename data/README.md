# `web_dashboard/data`

## 1. File explanation

- `dashboard_data_bundle.json`: Main frontend bundle with dashboard-ready summaries.
- `grid_predictions.csv`: Grid-level prediction table with labels, ranks, and opportunity scores.
- `grid_summary.csv`: City-level summary table for all grids.
- `kpi_summary.json`: KPI, freshness metadata, and best-model summary.
- `model_runs.csv`: Model run table with metrics, split strategy, and verification notes.
- `favicon.svg`: Dashboard icon file.

- `spatial/index.json`: Manifest file listing available spatial layers.
- `spatial/full/DaNang.geojson`: Full grid map layer for Da Nang.
- `spatial/full/HaNoi.geojson`: Full grid map layer for Ha Noi.
- `spatial/full/HCM.geojson`: Full grid map layer for HCM.
- `spatial/hotspots/DaNang.geojson`: Hotspot-only grid layer for Da Nang.
- `spatial/hotspots/HaNoi.geojson`: Hotspot-only grid layer for Ha Noi.
- `spatial/hotspots/HCM.geojson`: Hotspot-only grid layer for HCM.

- `boundaries/da-nang.geojson`: Simple city boundary for Da Nang.
- `boundaries/ha-noi.geojson`: Simple city boundary for Ha Noi.
- `boundaries/tp-hcm.geojson`: Simple city boundary for HCM.

- `population/danang_population.geojson`: Population density layer for Da Nang.
- `population/hcm_population.geojson`: Population density layer for HCM.
- `population/hn_population.geojson`: Population density layer for Ha Noi.
